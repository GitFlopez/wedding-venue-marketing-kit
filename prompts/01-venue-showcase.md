# Prompt 1: Venue Showcase & Brand Positioning (FREE)

## Purpose
Generate venue description copy, brand story, launch/relaunch campaign, vendor partnership outreach, and wedding show materials.

## When to Use
- New venue launching or relaunching
- Updating website copy and directory listings
- Preparing for a bridal expo or wedding show
- Reaching out to vendor partners for referral relationships

---

## The Prompt

```
You are an expert wedding venue copywriter specializing in luxury and boutique event spaces.

Generate a complete Venue Showcase & Brand Positioning package for:

Venue Name: [venue_name]
Location: [city_state]
Venue Type: [venue_type]
Capacity: [capacity_range]
Style/Aesthetic: [style_keywords]
Services: [services_included]
Starting Price: [price_range]
Signature Features: [unique_features]
Liquor License: [liquor_license]
Coordinator: [coordinator_name]
Target Couples: [target_couples]
Packages: [special_packages]

DELIVER ALL OF THE FOLLOWING:

**1. HERO VENUE DESCRIPTION (150 words)**
- Lead with the emotional experience, not logistics
- Weave in signature features naturally
- Close with a soft call-to-action (schedule a tour)
- Do NOT use superlatives without context ("best," "most beautiful") — use specific sensory details instead

**2. BRAND STORY (200 words)**
- Origin story or mission of the venue
- What makes this team different (coordinator philosophy, service ethos)
- Emotional payoff for couples who choose this venue
- First-person voice (owner/coordinator perspective)

**3. GRAND OPENING / RELAUNCH ANNOUNCEMENT (3 versions)**
A. Email to past inquiry leads (200 words): Subject line + body, soft announcement tone
B. Facebook/Instagram caption (100 words): Excitement + call-to-action (book a tour)
C. Press release snippet (150 words): AP-style, dateline, quote from coordinator, boilerplate

**4. VENDOR PARTNERSHIP OUTREACH EMAIL (150 words)**
- Target: local florists, photographers, caterers, officiants
- Mutual referral proposal
- Professional tone, specific value proposition for the vendor
- DO NOT promise exclusivity unless venue_services confirms it

**5. WEDDING SHOW / BRIDAL EXPO BOOTH COPY**
- Headline banner (10 words max)
- Three 25-word bullet points (what couples get)
- Table display card (50 words): emotional + practical
- Show special offer copy (limited-time incentive)

COMPLIANCE NOTES (apply to all output):
- If liquor_license is provided: include permit number in alcohol-mention context
- Do NOT use "guaranteed" for weather, vendor availability, or guest count
- Do NOT use comparative superlatives without qualification
- ADA: avoid copy implying venue is inaccessible; if accessibility not specified, omit accessibility claims
- FTC 2023: no fabricated testimonials; any "couples say" language must be clearly framed as example/hypothetical
```

## Example Input

```
Venue Name: Desert Rose Estate
Location: Henderson, NV
Venue Type: Indoor/outdoor luxury desert estate
Capacity: 50–300 guests
Style/Aesthetic: Desert-chic, romantic, modern rustic
Services: In-house catering, full open bar, bridal suite, day-of coordinator, 14 vendor-preferred list
Starting Price: Packages from $8,500
Signature Features: Panoramic Spring Mountains views, outdoor ceremony garden with live olive trees, covered heated patio, Edison-lit reception hall
Liquor License: Nevada Catering Liquor Permit #NV-CLP-20241184
Coordinator: Priya Vasquez, CWEP (Certified Wedding & Event Planner)
Target Couples: Las Vegas residents, destination couples from CA/UT/AZ, military families (Nellis AFB)
Packages: Elopement (2–10 guests, $2,500), Micro-Wedding (10–50, $5,500), Grand Affair (up to 300, from $14,000)
```

---

## Tips

- Paste your venue details directly into the bracketed fields
- The more specific your `unique_features`, the stronger the sensory copy
- Add `coordinator_name` credentials (CWEP, ABC, etc.) for authority copy
- For the vendor outreach email, specify the vendor type you're targeting first (florist, photographer, etc.)
